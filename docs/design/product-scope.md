# 产品定位与范围

## 定位

| 维度 | 选择 |
|------|------|
| 设备 | Insta360 **GO 3S** |
| 传输 | **WiFi 无线**（便利优先） |
| 用户 | 个人使用 |
| 发布 | **开源**（计划 MIT） |
| 平台 | **macOS** |

## 一句话

GO 3S 专用 WiFi 增量导入助手：连上相机热点后，把新 MP4/LRV 自动拉到 Mac，无需插线。

## MVP 功能（第一版）

- [ ] 检测 Mac 是否已连接 `GO 3S*.OSC` WiFi  
- [ ] 列相机文件列表（TCP 6666）  
- [ ] 对比本地索引，仅下载新文件  
- [ ] **MP4 + 同名 LRV 成组下载**  
- [ ] 进度、断点续传、取消  
- [ ] 完成后可选在 Finder 打开目标目录  

## 明确不做（v1）

- 拼接、剪辑、转码  
- 多相机同时连接  
- 自动替 Mac 切换 WiFi（系统限制）  
- 删除相机端文件（风险高，后续再议）  

## 与 USB 方案关系

USB U 盘模式更稳、更快，可作为 README 中推荐的备用路径；本产品差异化在 **无线便利**。
